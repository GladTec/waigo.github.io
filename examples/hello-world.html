Hello world

This simple example demonstrates the basics of a Waigo app. It uses a controller and a view to send 'Hello world' back to the browser.

Available files:

  • views/index.jade
  • controllers/main.js


    • views/index.jade

      doctype html
html
  head
    title Hello world
  body
    h1= text

    • controllers/main.js

      "use strict";


exports.index = function*(next) {
  yield* this.render('index', {
    text: 'Hello world!'
  });
};
