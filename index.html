What is Waigo?

Waigo is a Node.js framework for building scalable web applications. Waigo uses ES6 generators and an innovative architecture to improve upon the current generation of frameworks.

Easy

  • Built on Koa/Express middleware architecture.
  • Command-line client to automate tasks.

Elegant

  • No more callback hell. Generators and promises are in.
  • Minimal core framework, with no unneeded cruft.

Extendable

  • Easily override framework behaviour.
  • Create and share customizations as plugins.

Quick start

$ npm install -g waigo
$ waigo init
$ ./start-app.js

If you visit http://localhost:3000 in your browser you should see some welcome text.

You can now start adding controllers and models and develop your application.
