What is Waigo?

Waigo is a backend MVC framework for building scalable web applications.

Easy

  • Built on generator middleware architecture.
  • Command-line client to automate tasks.

Clean

  • No more callback hell. Generators and promises are in.
  • Minimal core framework, with no unneeded cruft.

Customizable

  • Easily override framework behaviour.
  • Create and share customizations as plugins.

Quick start

$ npm install -g waigo
$ waigo init
$ waigo start
