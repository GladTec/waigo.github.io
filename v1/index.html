What is Waigo?

Waigo is a low-level framework for building scalable web applications in Node.js. Waigo combines es6 generators with a flexible module loading mechanism and configurable architecture to enable you to build the application you need, without getting in your way.

Next-generation

Waigo builds on Koa, the ES6 successor to Express. Avoid callback hell, get better error handling and cleaner asynchronous code thanks to Promises and Generators.

Customizable

Almost every module file within the core framework can be overridden, allowing you to customise the behaviour of the framework to suit your needs. It gets out of your way when you need it to.

Lean and fast

The core framework is minimal and gives you a basic structure to develop your app with. Use your preferred back-end storage and front-end UI frameworks.

Web and mobile

Need a backend for your mobile app? With output formats you can re-use controllers to serve both web and REST API clients.

Quick setup

The command-line client makes it really easy to start using Waigo. The best part is you can extend it with your own commands relevant to your application.

Re-usable

Have an override or feature you'd like to share? Just bundle it up as a plugin. And since plugins are just NPM modules they're easy to distribute and use.

Quick start

$ npm install -g waigo
$ waigo init
$ ./start-app.js

If you visit http://localhost:3000 in your browser you should see some welcome text.

You can now start adding controllers and models and develop your application.
