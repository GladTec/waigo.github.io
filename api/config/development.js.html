Development mode configuration

This is the development mode configuration for the application.

This configuration module file is optional.

method

Option Type Description
config Object Configuration object to modify. 
module.exports = function(config) {
  
  config.middleware.options.errorHandler = {
    showStack: true
  }

logging

Logging config.

property

config.logging = {
    winston: {
      // log to console
      console: {
        // minimum level to log at
        level: 'debug',
        colorize: true,
        timestamp: true
      }
    }
  };
};
